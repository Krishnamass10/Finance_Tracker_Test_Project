# 💰 Finance Tracker

A full-stack finance management web application built using Django and MySQL.

## Features

- User Authentication
- Income & Expense Tracking
- Dashboard Analytics
- Pie Chart Reports
- Excel Export
- PDF Export
- Finance Chatbot
- Responsive Bootstrap UI

## Tech Stack

- Django
- MySQL
- Bootstrap
- JavaScript
- Chart.js
- openpyxl
- reportlab
